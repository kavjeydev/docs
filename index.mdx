## What is Trainly?

Trainly is a **GraphRAG (Graph Retrieval-Augmented Generation)** platform that enables you to build intelligent applications powered by your own documents. Upload files, and Trainly creates a searchable knowledge graph that AI models can query with accurate, citation-backed answers.

<Note>
  **Privacy-First Architecture**: User data is completely isolated. Developers get AI responses only‚Äînever raw files or documents.
</Note>

## Quick Start

Get up and running in 3 minutes:

<Steps>
  <Step title="Create an Account">
    Sign up at [trainlyai.com](https://trainlyai.com) and create your first chat
  </Step>
  <Step title="Upload Documents">
    Upload PDFs, documents, or text files to build your knowledge base
  </Step>
  <Step title="Get API Access">
    Enable API access in settings, publish your chat, and generate an API key
  </Step>
  <Step title="Start Building">
    Use our SDKs or REST API to integrate Trainly into your application
  </Step>
</Steps>

<CodeGroup>

```bash cURL
curl -X POST https://api.trainlyai.com/v1/chat_abc123/answer_question \
  -H "Authorization: Bearer tk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"question": "What are the main findings?"}'
```

## Example: Building a Document Q&A System

Here's a complete example in under 50 lines:

<CodeGroup>

```typescript Next.js
// app/api/query/route.ts
import { TrainlyClient } from "@trainly/react";
import { NextResponse } from "next/server";

const trainly = new TrainlyClient({
  apiKey: process.env.TRAINLY_API_KEY!,
  chatId: process.env.TRAINLY_CHAT_ID!,
});

export async function POST(request: Request) {
  const { question } = await request.json();

  const response = await trainly.query({
    question,
    model: "gpt-4o-mini",
  });

  return NextResponse.json({
    answer: response.answer,
    citations: response.context.slice(0, 5),
  });
}
```

```python FastAPI
from fastapi import FastAPI, HTTPException
from trainly import TrainlyClient
from pydantic import BaseModel

app = FastAPI()

trainly = TrainlyClient(
    api_key=os.getenv("TRAINLY_API_KEY"),
    chat_id=os.getenv("TRAINLY_CHAT_ID")
)

class QueryRequest(BaseModel):
    question: str

@app.post("/query")
async def query_documents(request: QueryRequest):
    try:
        response = trainly.query(
            question=request.question,
            model="gpt-4o-mini"
        )

        return {
            "answer": response.answer,
            "citations": response.context[:5]
        }
    except Exception as e:
        raise HTTPException(status_code=500, detail=str(e))
```

```javascript React
import { useState } from 'react';
import { TrainlyClient } from '@trainly/react';

function DocumentChat() {
  const [answer, setAnswer] = useState('');
  const [loading, setLoading] = useState(false);

  async function handleQuery(question) {
    setLoading(true);

    const response = await fetch('/api/query', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ question })
    });

    const data = await response.json();
    setAnswer(data.answer);
    setLoading(false);
  }

  return (
    <div>
      <input
        onKeyPress={(e) => e.key === 'Enter' && handleQuery(e.target.value)}
        placeholder="Ask a question..."
      />
      {loading && <div>Loading...</div>}
      {answer && <div>{answer}</div>}
    </div>
  );
}
```
